# Mini Search Engine - Phase 3

## Overview
Optimized Python mini search engine using an inverted index and TF-IDF for ranked keyword search. Phase 3 improves performance and scalability through efficient data structures and caching. 

## Features
- Inverted index + TF-IDF ranking
- Optimized postings using sets
- IDF caching for faster queries
- Stress-tested with large datasets
- Handles edge cases (empty, no results, punctuation)
